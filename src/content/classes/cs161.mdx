## Course Overview

The core algorithms course that every CS student should take. Changed how I think about problem-solving.

## Key Topics

### Divide and Conquer
- Master theorem
- Closest pair of points
- FFT

### Graph Algorithms
- BFS, DFS, topological sort
- Shortest paths (Dijkstra, Bellman-Ford)
- MST (Prim, Kruskal)
- Network flow

### Dynamic Programming
- Optimal substructure
- Edit distance
- Knapsack problems

### NP-Completeness
- P vs NP
- Reductions
- Approximation algorithms
