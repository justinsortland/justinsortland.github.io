## Course Overview

The classic Stanford ML course that provides rigorous mathematical foundations for machine learning algorithms.

## Key Topics

### Supervised Learning
- Linear regression, logistic regression
- Generalized linear models
- SVMs and kernel methods
- Neural networks

### Learning Theory
- Bias-variance decomposition
- VC dimension
- PAC learning

### Unsupervised Learning
- K-means, EM algorithm
- PCA, ICA
- Density estimation

## Mathematical Foundations

The course emphasizes deriving algorithms from first principles. For example, logistic regression emerges naturally from assuming exponential family distributions.
