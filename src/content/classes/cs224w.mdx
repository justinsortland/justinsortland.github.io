## Course Overview

CS 224W covers the foundations and frontiers of machine learning on graphs. The course assumes familiarity with ML basics and linear algebra.

## Key Topics

### Node Embeddings
- DeepWalk and Node2Vec
- Matrix factorization perspectives
- Scalability considerations

### Graph Neural Networks
- GCN, GraphSAGE, GAT
- Expressiveness and WL test
- Over-smoothing and over-squashing

### Knowledge Graphs
- Translational embeddings (TransE, RotatE)
- Reasoning and link prediction
- Temporal knowledge graphs

## Projects

Built a recommendation system using GraphSAGE for a movie dataset, achieving 15% improvement over collaborative filtering baselines.
