## Overview

Spectral clustering leverages the eigenstructure of similarity graphs to find clusters, but the computational cost scales cubically with dataset size. SpectralKit makes spectral clustering practical for large-scale applications.

## Mathematical Foundation

Given a similarity matrix $W$, spectral clustering computes:

1. The graph Laplacian: $L = D - W$ where $D_{ii} = \sum_j W_{ij}$
2. The $k$ smallest eigenvectors of $L$
3. K-means clustering on the eigenvector embedding

### Nyström Approximation

Instead of computing the full kernel matrix, we sample $m \ll n$ landmark points:

$$
\tilde{K} = K_{nm} K_{mm}^{-1} K_{mn}
$$

This reduces memory from $O(n^2)$ to $O(nm)$.

### Randomized SVD

For the eigendecomposition, we use randomized algorithms:

```python
from spectralkit import randomized_eigh

# Compute top-k eigenvectors in O(nk²) time
eigenvalues, eigenvectors = randomized_eigh(
    laplacian,
    k=10,
    oversampling=5,
    power_iterations=2
)
```

## Usage

```python
from spectralkit import SpectralClustering

clusterer = SpectralClustering(
    n_clusters=10,
    n_landmarks=1000,
    affinity='rbf',
    gamma=0.1,
    device='cuda'
)

labels = clusterer.fit_predict(X)
```
