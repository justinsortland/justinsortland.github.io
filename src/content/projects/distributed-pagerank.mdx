## Overview

PageRank remains one of the most important graph algorithms, used beyond web ranking for recommendation systems, fraud detection, and influence analysis. This project implements production-grade PageRank at web scale.

## The Algorithm

PageRank iteratively computes importance scores:

$$
PR(v) = \frac{1-d}{N} + d \sum_{u \in \text{in}(v)} \frac{PR(u)}{|\text{out}(u)|}
$$

where $d \approx 0.85$ is the damping factor.

### Delta-Based Updates

Instead of recomputing full PageRank values, we propagate deltas:

```scala
def computePageRank(graph: Graph[Double, Double], 
                    damping: Double = 0.85,
                    tolerance: Double = 1e-6): Graph[Double, Double] = {
  
  val initialGraph = graph.mapVertices((_, _) => 1.0 / graph.numVertices)
  
  def sendMessage(edge: EdgeTriplet[Double, Double]): Iterator[(VertexId, Double)] = {
    val contribution = edge.srcAttr / edge.attr
    Iterator((edge.dstId, contribution))
  }
  
  Pregel(initialGraph, 0.0, maxIterations = 100)(
    vprog = (_, attr, msg) => (1 - damping) / graph.numVertices + damping * msg,
    sendMsg = sendMessage,
    mergeMsg = _ + _
  )
}
```
